# MagicPlayer
基于ijkPlayer的视频播放器

## Get Start

### gradle
Step 1.Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        compile 'com.github.DevlinChiu:MagicPlayer:v1.0'
	}
